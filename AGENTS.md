## Agent skills

### Issue tracker

Issue 和 PRD 使用 GitHub Issues 跟踪，并通过 `gh` CLI 操作。详见 `docs/agents/issue-tracker.md`。

### Triage labels

本仓库使用默认五类 triage 标签。详见 `docs/agents/triage-labels.md`。

### Domain docs

本仓库使用 single-context 领域文档布局。详见 `docs/agents/domain.md`。

### Language

后续开发以中文为主。面向用户的 UI 文案、README、Issue/PR 描述、测试用例名称和项目文档优先使用中文；代码标识符、API 字段名、环境变量名、依赖配置和第三方工具命令保持英文。
