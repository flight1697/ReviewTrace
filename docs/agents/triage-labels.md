# Triage 标签

工程技能内部使用五个标准 triage 角色。这个文件负责把这些角色映射到本仓库 issue tracker 里实际使用的标签字符串。

| mattpocock/skills 中的标签 | 本仓库使用的标签 | 含义 |
| -------------------------- | ---------------- | ---- |
| `needs-triage`             | `needs-triage`   | 维护者需要评估这个 issue |
| `needs-info`               | `needs-info`     | 等待提交者补充更多信息 |
| `ready-for-agent`          | `ready-for-agent` | 规格已经清楚，可以交给 agent 实现 |
| `ready-for-human`          | `ready-for-human` | 需要人工实现 |
| `wontfix`                  | `wontfix`        | 不会处理 |

当某个 skill 提到一个角色时，例如“应用 AFK-ready triage 标签”，就使用这个表格中对应的标签字符串。

如果之后仓库改用其他标签体系，只需要修改右侧“本仓库使用的标签”这一列。
